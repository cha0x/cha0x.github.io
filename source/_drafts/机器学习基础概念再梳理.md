# 机器学习
何谓学习？
**学习=表示+评价+优化**

| 表示      | 评价             | 优化    |
| ------- | -------------- | ----- |
| 基于实例的方法 | 准确/错误比率        | 组合优化  |
| 近邻方法    | 精确率和召回率        | 贪心搜索  |
| 支持向量机   | 平方误差           | 柱搜索   |
| 超平面方法   | 似然(likelihood) | 分支限界法 |
| 朴素贝叶斯   | 后验概率           | 连续优化  |
| 逻辑斯蒂回归  | 信息增益           | 无约束   |
| 决策树方法   | K-L距离          | 梯度下降  |
| 规则集的方法  | 成本/效用          | 共轭梯度  |
| 命题规则    | 利润             | 拟牛顿法  |
| 逻辑程序    |                | 有约束   |
| 神经网络    |                | 线性规划  |
| 图模型     |                | 二次规划  |
| 贝叶斯网络   |                |       |
| 条件随机场   |                |       |
